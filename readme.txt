== Description ==

Best Review Plugin. Customer reviews powered with shortcode to display both grid reviews and slider reviews.

== Features ==

1. Display all client reviews with count functionality
* [review_grid]
* [review_grid count="9"]

2. Display client reviews filtered by Category with count functionality
* [review_grid_cat category="test"]
* [review_grid_cat category="test" count="9"]

3. Display client slider reviews with count functionality
* [client_slider]
* [client_slider count="10"]

4. Add Review metaboxes;
* source (facebook, google and tripadvisor)
* company name
* position
* address
* blurb


== Installation ==

1. In your admin panel, go to Plugins -> click the 'Add New' button.
2. Click 'Upload Plugin', choose file and install.
3. Click on the 'Activate' button to use wp reviews right away.
